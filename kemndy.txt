1. Zainstaluj wymagane narzędzia
ImageMagick: Jest wymagany do przetwarzania obrazów, takich jak tła. Zainstaluj go za pomocą:

bash
sudo pacman -S imagemagick
GRUB: Upewnij się, że GRUB jest zainstalowany i skonfigurowany w systemie:

bash
sudo pacman -S grub
2. Powiększenie tekstu w GRUB
Zmień rozdzielczość GRUB, aby tekst był większy:

Edytuj plik /etc/default/grub:

bash
sudo nano /etc/default/grub
Znajdź linię GRUB_GFXMODE i ustaw odpowiednią rozdzielczość, np.:

bash
GRUB_GFXMODE=1920x1080x32
Dodaj lub zmodyfikuj linię:

bash
GRUB_FONT=/boot/grub/fonts/DejaVuSansMono.pf2
Jeśli chcesz zmienić czcionkę, użyj narzędzia grub-mkfont, aby przekonwertować czcionkę na format GRUB:

bash
sudo grub-mkfont -s 24 -o /boot/grub/fonts/DejaVuSansMono.pf2 /ścieżka/do/DejaVuSansMono.ttf
Zaktualizuj konfigurację GRUB:

bash
sudo grub-mkconfig -o /boot/grub/grub.cfg
3. Dodanie niestandardowego tła
Przygotuj obraz tła w odpowiedniej rozdzielczości (np. 1920x1080 dla Full HD).

Umieść obraz w katalogu projektu i nazwij go background.jpg.

Uruchom instalator z odpowiednimi opcjami, np.:

bash
sudo ./install.sh -t mountain -s 1080p
Jeśli chcesz ręcznie ustawić tło, edytuj plik /etc/default/grub i dodaj:

bash
GRUB_BACKGROUND=/ścieżka/do/background.jpg
Następnie zaktualizuj GRUB:

bash
sudo grub-mkconfig -o /boot/grub/grub.cfg
4. Uruchomienie instalatora
Po upewnieniu się, że wszystkie zależności są zainstalowane, uruchom instalator z wybranymi opcjami:

bash
sudo ./install.sh -t mountain -s 2k
